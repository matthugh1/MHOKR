# OKR Nexus Documentation

This directory contains the comprehensive user guide for OKR Nexus.

## Structure

```
/docs
├── layout.tsx          # Main layout with navigation and search
├── page.tsx            # Documentation homepage/index
├── getting-started/    # Getting started guide
├── dashboard/          # Dashboard guide
├── okr-management/     # OKR Management guide
├── visual-builder/     # Visual Builder guide
├── check-ins/          # Check-ins guide
├── ai-assistant/       # AI Assistant guide
├── analytics/          # Analytics & Reporting guide
├── settings/           # Settings & Administration guide
├── SCREENSHOTS.md      # Screenshot capture guide
└── README.md           # This file
```

## Features

### Navigation
- Sticky header with search
- Sidebar navigation (desktop)
- Top navigation (mobile/tablet)
- Breadcrumb navigation
- Related guides links

### Search
- Real-time search as you type
- Searches titles and keywords
- Dropdown results with preview
- Click to navigate

### Content
- Comprehensive guides for all major features
- Step-by-step instructions
- Best practices and tips
- Related guides cross-references
- Organized by feature area

## Accessing Documentation

Users can access the documentation by:
1. Clicking "Documentation" on the home page
2. Navigating to `/docs` directly
3. Using the link in the application navigation (if added)

## Adding Screenshots

See `SCREENSHOTS.md` for instructions on capturing and adding screenshots to the documentation.
